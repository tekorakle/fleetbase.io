# Core Platform

The Fleetbase core platform is the foundation that underpins every module and extension. It handles the concerns that every serious logistics system needs: multi-tenant organisation management, role-based access control, a real-time event bus, a REST API, and a developer console for managing integrations.

Start with the [Platform Overview](/docs/platform/getting-started/overview) for a complete introduction, or jump straight to a quickstart if you already know what you need.
