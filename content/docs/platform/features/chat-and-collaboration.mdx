# Chat & Collaboration

Logistics is inherently collaborative. Dispatchers need to communicate with drivers, customer support needs to update warehouse staff, and managers need to leave notes on specific orders.

Instead of relying on external tools like Slack or WhatsApp, Fleetbase includes a built-in chat and commenting system that keeps communication directly tied to the relevant operational data.

## Contextual Comments

The most powerful collaboration feature in Fleetbase is **Contextual Comments**. You can leave comments directly on almost any resource in the platform, such as an Order, a Driver profile, or a Vehicle record.

This ensures that anyone viewing that resource has the full context of what has happened, without needing to search through external chat logs.

### Leaving a Comment

To leave a comment on a resource:

1. Open the resource (e.g., click on an Order in the FleetOps dashboard).
2. Look for the **Comments** or **Activity** tab in the resource's detail view.
3. Type your message in the input box.
4. Click **Post Comment**.

Your comment will be saved with a timestamp and your name, creating a permanent, chronological record of the conversation.

### Mentions (Coming Soon)

In a future update, you will be able to `@mention` other users in your organization within a comment. This will trigger a notification for that user, drawing their attention to the specific resource.

## Direct Chat

For general communication that isn't tied to a specific resource, Fleetbase provides a direct chat system.

### Using the Chat Interface

1. Click the **Chat** icon in the top navigation bar of the Fleetbase Console.
2. This opens the chat panel, where you can see your recent conversations.
3. Click on a user's name to open a direct message thread with them.
4. Type your message and press Enter to send.

The chat system uses the same real-time WebSocket infrastructure as the rest of the platform, ensuring messages are delivered instantly.

### Group Chat (Coming Soon)

Currently, the chat system supports direct 1-on-1 messaging. In a future update, this will be expanded to support Group Chats, allowing you to create dedicated channels for specific teams (e.g., "North America Dispatchers") or topics (e.g., "Urgent Issues").

## Driver Communication

If you are using the **FleetOps** extension and the **Navigator** mobile app, the chat system extends to your drivers on the road.

Dispatchers can use the Fleetbase Console to send direct messages to drivers. These messages appear as push notifications on the driver's mobile device, and the driver can reply directly from the Navigator app.

This provides a secure, centralized, and fully logged communication channel between the back office and the field.
