# Platform Features

Because Fleetbase is built as an operating system, it provides a suite of core features that are available across all installed extensions. These built-in capabilities ensure a consistent user experience and reduce the need for extensions to reinvent the wheel.

Whether you are managing a fleet in FleetOps, processing orders in Storefront, or tracking inventory in Pallet, you can leverage these platform-wide features to customize your workflows and improve collaboration.

## Core Capabilities

### [Custom Fields](/docs/platform/features/custom-fields)
Every business is unique. Custom Fields allow you to add your own data attributes to any resource in Fleetbase (like Orders, Drivers, or Customers) without writing any code or modifying the database schema.

### [Dashboards & Widgets](/docs/platform/features/dashboards-and-widgets)
Gain full visibility into your operations at a glance. The Fleetbase Console allows you to create custom dashboards and populate them with widgets provided by your installed extensions.

### [Chat & Collaboration](/docs/platform/features/chat-and-collaboration)
Logistics is a team effort. Fleetbase includes a built-in chat system and commenting functionality, allowing your team to communicate directly within the context of specific orders, vehicles, or issues.

### [Notifications](/docs/platform/features/notifications)
Keep your team and your customers informed. The notification system handles sending alerts via email, SMS, and push notifications based on real-time events.

### [Scheduling](/docs/platform/features/scheduling)
Automate your operations. The scheduling engine allows you to set up recurring tasks, automated reports, and time-based triggers across the platform.
