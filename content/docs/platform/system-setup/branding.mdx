# Branding

The Branding settings let you replace the default Fleetbase logo and icon with your own, and set the default color theme for the console. Navigate to **Admin → Branding** to access these settings.

![Branding settings — icon upload, logo upload, and default theme selector](/images/screenshots/admin/admin-branding.webp)

## Icon

The icon is the small square image shown in the browser tab, mobile app launcher, and compact views throughout the console. Upload a square PNG or SVG — 32×32 px is the display size, but uploading at 2× (64×64 px) or higher gives a sharper result on high-DPI screens.

Click **Upload New** to select an image from your machine. The icon is uploaded immediately and the preview updates in place. Click **Reset to Default** to remove a custom icon and restore the Fleetbase default.

## Logo

The logo appears in the console sidebar header and on the login screen. The display area is approximately 160×56 px; a transparent-background PNG or SVG at 2× that size works best.

Click **Upload New** to upload a replacement logo. Click **Reset to Default** to restore the Fleetbase logo.

## Default Theme

Sets the color theme applied to the console for all users on their first login. Users can override this individually from their personal settings.

| Option | Description |
| :--- | :--- |
| **Light** | White background, dark text — default |
| **Dark** | Dark background, light text |

## Saving

Click **Save Changes** in the page header to apply all branding updates. Changes take effect immediately — no restart required.
