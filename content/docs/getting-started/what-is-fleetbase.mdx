# What is Fleetbase?

Fleetbase is an open-source, modular logistics and supply chain operating system designed to streamline management, planning, optimization, and operational control across various sectors of the supply chain industry. Built with extensibility at its core, Fleetbase provides a powerful foundation for organizations to manage complex logistics operations while offering developers the flexibility to build custom solutions.

Unlike traditional monolithic logistics software, Fleetbase operates as a true "operating system" for logistics. It provides the core infrastructure—identity management, real-time events, API gateways, and a unified user interface—while specific logistics capabilities are added through modular extensions.

## Who is Fleetbase For?

Fleetbase is designed to serve a wide range of industries and use cases, bridging the gap between operational requirements and developer flexibility.

| Industry / Role | Primary Use Cases |
|---|---|
| **E-commerce & Retail** | Manage deliveries, track orders in real-time, and optimize last-mile logistics for customer shipments. |
| **Food & Beverage** | Coordinate restaurant deliveries, manage driver fleets, and track real-time order status from kitchen to customer. |
| **Courier Services** | Dispatch drivers, optimize multi-stop routes, and provide customers with live tracking links. |
| **Field Services** | Schedule technicians, manage geographic service areas, and track job completion and work orders. |
| **Enterprise Logistics** | Build custom supply chain solutions, integrate with existing ERPs, and manage complex multi-tenant operations. |
| **Developers** | Extend the platform with custom extensions, integrate via the comprehensive REST API, and build bespoke logistics applications. |

## Key Features

Fleetbase provides a comprehensive suite of features designed to support modern logistics operations:

- **Extensible Architecture**: Build installable extensions and add functionality directly into the OS via a modular architecture.
- **Developer Friendly**: A comprehensive RESTful API, real-time WebSocket events, and webhooks make it seamless to integrate with external systems or develop custom applications.
- **Native Mobile Apps**: A collection of open-source, native mobile applications designed for both operations (drivers) and customers.
- **Collaboration**: A dedicated chat and comments system enables real-time collaboration across your organization.
- **Security & IAM**: Secure data encryption, adherence to industry-standard security practices, and a comprehensive dynamic Identity and Access Management (IAM) system.
- **Telematics Integration**: Connect hardware devices and sensors to provide real-time feedback and visibility into fleet operations.
- **Dynamic Automation**: Configurable rules, flows, and logic enable automation and customization of operational workflows.
- **Custom Dashboards**: Create custom dashboards and widgets to gain full visibility into operations at a glance.

## The Fleetbase Ecosystem

The Fleetbase ecosystem consists of several core components that work together to provide a complete logistics solution:

1. **The Core Platform**: The underlying operating system that handles authentication, API routing, real-time events, and extension management.
2. **Extensions**: Modular packages that add specific functionality, such as FleetOps (fleet management), Storefront (headless e-commerce), and Ledger (accounting).
3. **Mobile Apps**: Open-source companion apps, including the Navigator App for drivers and the Storefront App for customers.
4. **The Developer Console**: A built-in suite of tools for managing API keys, webhooks, and real-time socket events.

Whether you are a logistics operator looking for a modern management platform, or a developer building the next generation of supply chain software, Fleetbase provides the tools and infrastructure you need to succeed.
