# Quickstart (Cloud)

Fleetbase Cloud is the fastest way to get started with Fleetbase. It is a fully managed, hosted version of the Fleetbase operating system, allowing you to focus on your logistics operations and integrations without worrying about infrastructure, scaling, or maintenance.

## Step 1: Create an Account

To get started, you need to create a Fleetbase Cloud account:

1. Navigate to the [Fleetbase Console](https://console.fleetbase.io).
2. Click on **Sign Up** and provide your email address and a secure password.
3. Verify your email address using the link sent to your inbox.
4. Log in to the console.

## Step 2: Set Up Your Organization

Upon your first login, you will be prompted to set up your organization. This represents your company or primary operational entity within Fleetbase.

1. Enter your **Organization Name**.
2. Provide any additional requested details (such as industry or primary use case).
3. Click **Create Organization**.

Your organization is now created, and you will be redirected to the main Fleetbase dashboard.

## Step 3: Install Extensions

By default, the core Fleetbase platform provides essential services like IAM and the Developer Console. To add logistics capabilities, you need to install extensions.

1. In the left sidebar, navigate to **Extensions**.
2. Browse the available extensions in the marketplace.
3. For fleet management and dispatch, locate **FleetOps** and click **Install**.
4. For headless e-commerce, locate **Storefront** and click **Install**.

Once installed, the extensions will automatically add their respective menus and features to your console.

## Step 4: Generate API Credentials

If you plan to integrate Fleetbase with external systems or build custom applications, you will need API credentials.

1. In the left sidebar, navigate to **Developers** > **API Keys**.
2. Click **Create API Key**.
3. Provide a name for the key (e.g., "ERP Integration").
4. Select the appropriate environment (Test or Live).
5. Copy the generated API key and store it securely. You will use this as a Bearer token in your API requests.

## Next Steps

With your Fleetbase Cloud instance set up, you can now begin configuring your operations:

- **Managing a Fleet?** Head over to the [FleetOps Documentation](/docs/fleetops) to learn how to set up drivers, vehicles, and service rates.
- **Building a Marketplace?** Check out the [Storefront Documentation](/docs/storefront) to start building your product catalog.
- **Integrating Systems?** Review the [API Reference](/docs/api-reference) to understand how to interact with Fleetbase programmatically.
