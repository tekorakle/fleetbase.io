# Commands

Quick reference for every command the `flb` CLI exposes. Click any command for the full options, examples, and behavior.

## Install Fleetbase

| Command | Description |
| :--- | :--- |
| [`flb install-fleetbase`](/docs/cli/installation/overview) | Interactive Docker-based wizard that installs a self-hosted Fleetbase instance |

## Account

| Command | Description |
| :--- | :--- |
| [`flb register`](/docs/cli/account/register) | Create a new Fleetbase Extension Registry developer account |
| [`flb verify`](/docs/cli/account/verify) | Verify a registry account email and receive your auth token |
| [`flb resend-verification`](/docs/cli/account/resend-verification) | Request a fresh verification code |
| [`flb login`](/docs/cli/account/login) | Log in to the registry with username + password (writes to `~/.npmrc`) |
| [`flb set-auth`](/docs/cli/account/set-auth) | Write a registry auth token into a Fleetbase instance's `console/.npmrc` and `api/auth.json` |
| [`flb generate-token`](/docs/cli/account/generate-token) | Generate (or regenerate) a long-lived registry auth token |

## Manage Extensions

| Command | Description |
| :--- | :--- |
| [`flb search`](/docs/cli/extensions/search) | Browse and filter the Extension Registry from the command line |
| [`flb install`](/docs/cli/extensions/install) | Install an extension into a running Fleetbase instance |
| [`flb uninstall`](/docs/cli/extensions/uninstall) | Remove an installed extension from a Fleetbase instance |

## Publish Extensions

| Command | Description |
| :--- | :--- |
| [`flb scaffold`](/docs/cli/extension-development/scaffold) | Generate a new extension from the starter template |
| [`flb version-bump`](/docs/cli/extension-development/version-bump) | Increment the version across `extension.json`, `package.json`, and `composer.json` |
| [`flb bundle`](/docs/cli/extension-development/bundle) | Pack an extension into a versioned `.tar.gz` |
| [`flb bundle-upload`](/docs/cli/extension-development/bundle-upload) | Upload an existing bundle to the registry |
| [`flb publish`](/docs/cli/extension-development/publish) | Publish an extension to the registry via `npm publish` |
| [`flb unpublish`](/docs/cli/extension-development/unpublish) | Remove a published extension from the registry |

## Built-in

| Command | Description |
| :--- | :--- |
| `flb --version` (or `-v`) | Print the CLI version |
| `flb --help` | Show top-level help and command list |
| `flb <command> --help` | Show help for a specific command |

## Global Options

These options work with every command:

| Option | Default | Description |
| :--- | :--- | :--- |
| `-v, --version` | — | Print the CLI version |
| `-r, --registry [url]` | `https://registry.fleetbase.io` | Override the registry URL |
| `--help` | — | Show help for the current command |
