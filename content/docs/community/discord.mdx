# Discord Community

The Fleetbase Discord server is the best place to get help, share your projects, and connect with other developers and the core team.

## Join the Server

Click the link below to join the Fleetbase Discord server:

**[Join the Fleetbase Discord &rarr;](https://discord.gg/fleetbase)**

## Channels

Once you join, you'll find several channels dedicated to different topics:

- `#general`: General discussion about Fleetbase, logistics, and supply chain tech.
- `#help`: Get help with installation, configuration, and API usage.
- `#development`: Discuss extension development, the Universe Service, and contributing to the core platform.
- `#showcase`: Share what you've built with Fleetbase!

## Guidelines

To keep the community welcoming and productive, please follow these guidelines:

1. **Be respectful**: Treat everyone with respect. Harassment, hate speech, and toxic behavior will not be tolerated.
2. **Search before asking**: Your question might have already been answered. Use the Discord search bar or check the [FAQ](/docs/community/faq) before posting.
3. **Provide context**: When asking for help, provide as much context as possible (e.g., error messages, code snippets, Fleetbase version).
4. **No spam**: Please do not spam the channels with promotional content or irrelevant links.
