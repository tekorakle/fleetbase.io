# Community & Support

Welcome to the Fleetbase community! Whether you're looking for help, want to contribute to the open-source project, or just want to connect with other developers, you're in the right place.

## Get Help

- **[Discord Community](/docs/community/discord)**: Join our active Discord server to ask questions, share your projects, and chat directly with the core team.
- **[FAQ](/docs/community/faq)**: Check out our Frequently Asked Questions for quick answers to common issues.
- **GitHub Issues**: If you've found a bug or have a feature request, please open an issue on the relevant [GitHub repository](https://github.com/fleetbase).

## Contribute

Fleetbase is open-source, and we welcome contributions from the community!

- **[Contributing Guide](/docs/community/contributing)**: Learn how to set up your development environment, submit pull requests, and follow our coding standards.
- **Good First Issues**: Look for issues tagged with `good first issue` on GitHub if you're looking for a place to start.

## Stay Updated

- **[Changelog](/docs/community/changelog)**: Keep track of new features, bug fixes, and breaking changes across the Fleetbase ecosystem.
- **Twitter/X**: Follow us [@fleetbase_io](https://twitter.com/fleetbase_io) for announcements and updates.
