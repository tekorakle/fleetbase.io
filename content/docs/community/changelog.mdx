# Changelog

Stay up to date with the latest Fleetbase releases, features, and bug fixes.

## Where to find release notes

Because Fleetbase is a modular platform composed of many different repositories (Core API, Console, FleetOps, Storefront, Mobile Apps), we maintain separate changelogs for each project on GitHub.

You can find the latest release notes and changelogs on the **Releases** page of each repository:

- **[Fleetbase Core Releases](https://github.com/fleetbase/fleetbase/releases)**: Updates to the core platform, API, and Console.
- **[FleetOps Releases](https://github.com/fleetbase/fleetops/releases)**: Updates to the FleetOps extension.
- **[Storefront Releases](https://github.com/fleetbase/storefront/releases)**: Updates to the Storefront extension.
- **[Navigator App Releases](https://github.com/fleetbase/navigator-app/releases)**: Updates to the driver mobile app.
- **[Storefront App Releases](https://github.com/fleetbase/storefront-app/releases)**: Updates to the customer mobile app.

## Subscribing to updates

To get notified when a new version is released:

1. Go to the relevant GitHub repository.
2. Click the **Watch** button in the top right corner.
3. Select **Custom** > **Releases**.

You will now receive an email notification whenever a new release is published.

## Major Announcements

For major platform updates, new extension releases, and company news, follow us on [Twitter/X](https://twitter.com/fleetbase_io) or join the `#announcements` channel in our [Discord Community](/docs/community/discord).
