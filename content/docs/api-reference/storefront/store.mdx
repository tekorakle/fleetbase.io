# Store

The **Store** resource provides information about the store or network that the current API key is associated with.

This includes the store's name, description, logo, contact information, and operating hours.

## The Store Object

```json
{
  "id": "store_xxxxxxxx",
  "name": "Acme Electronics",
  "description": "The best electronics store in town.",
  "logo_url": "https://example.com/logo.png",
  "backdrop_url": "https://example.com/backdrop.png",
  "phone": "+14155551234",
  "email": "hello@acme.com",
  "website": "https://acme.com",
  "facebook": "acmeelectronics",
  "instagram": "acmeelectronics",
  "twitter": "acmeelectronics",
  "currency": "USD",
  "options": {
    "tax_included": true,
    "require_phone": false
  },
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The name of the store. |
| `description` | string | A description of the store. |
| `logo_url` | string | The URL of the store's logo image. |
| `backdrop_url` | string | The URL of the store's backdrop or cover image. |
| `phone` | string | The store's contact phone number. |
| `email` | string | The store's contact email address. |
| `website` | string | The store's website URL. |
| `currency` | string | The default currency used by the store (e.g., `USD`). |
| `options` | object | A set of configuration options for the store. |

## Retrieve Store Information

Retrieves the details of the store associated with the provided API key.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/about \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List Store Locations

Returns a list of physical locations associated with the store.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/locations \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List Payment Gateways

Returns a list of active payment gateways configured for the store.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/gateways \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
