# Checkout

The **Checkout** resource is used to convert a [Cart](/docs/api-reference/storefront/cart) into an [Order](/docs/api-reference/fleetops/orders) and process payment.

Storefront supports multiple payment gateways, including Stripe, Cash on Delivery (COD), and custom gateways.

## The Checkout Object

```json
{
  "id": "checkout_xxxxxxxx",
  "cart": "cart_xxxxxxxx",
  "customer": "customer_xxxxxxxx",
  "amount": 19900,
  "currency": "USD",
  "status": "pending",
  "payment_method": "stripe",
  "payment_intent": "pi_xxxxxxxx",
  "client_secret": "pi_xxxxxxxx_secret_xxxxxxxx",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `cart` | string | The ID of the [Cart](/docs/api-reference/storefront/cart) being checked out. |
| `customer` | string | The ID of the [Customer](/docs/api-reference/storefront/customers) checking out. |
| `amount` | integer | The total amount to be charged (in cents). |
| `currency` | string | The currency of the checkout (e.g., `USD`). |
| `status` | string | The current status of the checkout (e.g., `pending`, `completed`, `failed`). |
| `payment_method` | string | The selected payment method (e.g., `stripe`, `cod`). |
| `payment_intent` | string | The ID of the Stripe Payment Intent (if applicable). |
| `client_secret` | string | The Stripe client secret used to confirm the payment on the frontend. |

## Initialize a Checkout

Initializes a new checkout session for a cart. This will calculate the final total (including taxes and shipping) and generate a payment intent if using Stripe.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/storefront/checkout \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "cart": "cart_xxxxxxxx",
    "customer": "customer_xxxxxxxx",
    "payment_method": "stripe",
    "shipping_address": "place_xxxxxxxx",
    "billing_address": "place_yyyyyyyy"
  }'
```

## Capture a Checkout

Captures the payment and converts the cart into an order. This should be called after the payment has been successfully confirmed on the frontend (e.g., via Stripe Elements).

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/storefront/checkout/capture \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "checkout": "checkout_xxxxxxxx",
    "payment_intent": "pi_xxxxxxxx"
  }'
```
