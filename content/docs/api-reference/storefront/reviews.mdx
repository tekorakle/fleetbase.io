# Reviews

A **Review** represents a customer's rating and feedback for a specific [Product](/docs/api-reference/storefront/products) or the Store itself.

## The Review Object

```json
{
  "id": "review_xxxxxxxx",
  "subject_uuid": "product_xxxxxxxx",
  "subject_type": "storefront:product",
  "customer": "customer_xxxxxxxx",
  "rating": 5,
  "content": "These headphones are amazing! The noise cancellation is top-notch.",
  "status": "published",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `subject_uuid` | string | The ID of the object being reviewed (e.g., a Product ID). |
| `subject_type` | string | The type of object being reviewed (e.g., `storefront:product`, `storefront:store`). |
| `customer` | string | The ID of the [Customer](/docs/api-reference/storefront/customers) who wrote the review. |
| `rating` | integer | The rating given by the customer (typically 1-5). |
| `content` | string | The text content of the review. |
| `status` | string | The current status of the review (e.g., `published`, `pending`, `rejected`). |

## Create a Review

Creates a new Review object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/storefront/reviews \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Customer-Token: session_xxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "subject_uuid": "product_xxxxxxxx",
    "subject_type": "storefront:product",
    "rating": 5,
    "content": "These headphones are amazing! The noise cancellation is top-notch."
  }'
```

## Retrieve a Review

Retrieves the details of an existing Review.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/reviews/review_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Reviews

Returns a list of Reviews. The reviews are returned sorted by creation date, with the most recently created reviews appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/reviews \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

### Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `subject_uuid` | string | Filter reviews by the ID of the object being reviewed. |
| `subject_type` | string | Filter reviews by the type of object being reviewed. |
| `customer` | string | Filter reviews by the ID of the customer who wrote them. |
