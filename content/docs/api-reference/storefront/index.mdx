# Storefront API

The **Storefront API** provides access to all resources related to headless e-commerce, including products, categories, carts, checkout, and customers.

## Base URL

All Storefront API requests should be made to the following base URL:

```text
https://api.fleetbase.io/v1/storefront
```

*(If you are self-hosting Fleetbase, replace `api.fleetbase.io` with your own domain).*

## Authentication

All requests to the Storefront API must be authenticated using a **Public Key**.

You can generate a Public Key in the [Developer Console](/docs/platform/developer-console/api-keys).

Pass the Public Key in the `Authorization` header as a Bearer token:

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/products \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Available Resources

### Catalog
- **[Store](/docs/api-reference/storefront/store)**: Retrieve information about the store or network.
- **[Products](/docs/api-reference/storefront/products)**: Browse and search the product catalog.
- **[Categories](/docs/api-reference/storefront/categories)**: Browse product categories and collections.

### Shopping
- **[Cart](/docs/api-reference/storefront/cart)**: Manage shopping carts and cart items.
- **[Checkout](/docs/api-reference/storefront/checkout)**: Process checkouts and create orders.

### Customers
- **[Customers](/docs/api-reference/storefront/customers)**: Manage customer accounts, addresses, and authentication.
- **[Reviews](/docs/api-reference/storefront/reviews)**: Read and submit product or store reviews.
