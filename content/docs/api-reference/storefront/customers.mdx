# Customers

A **Customer** represents a user who has registered an account on your Storefront.

Customers can save their addresses, view their order history, and manage their payment methods.

## The Customer Object

```json
{
  "id": "customer_xxxxxxxx",
  "name": "Jane Doe",
  "email": "jane.doe@example.com",
  "phone": "+14155553333",
  "status": "active",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The full name of the customer. |
| `email` | string | The customer's email address. |
| `phone` | string | The customer's phone number. |
| `status` | string | The current status of the customer account (e.g., `active`, `inactive`). |

## Register a Customer

Registers a new Customer account.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/storefront/customers \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane.doe@example.com",
    "phone": "+14155553333",
    "password": "securepassword123"
  }'
```

## Authenticate a Customer

Authenticates a customer and returns a session token.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/storefront/customers/login \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "jane.doe@example.com",
    "password": "securepassword123"
  }'
```

## Retrieve the Current Customer

Retrieves the details of the currently authenticated Customer.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/storefront/customers/me \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Customer-Token: session_xxxxxxxx"
```

## Update the Current Customer

Updates the details of the currently authenticated Customer.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/storefront/customers/me \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Customer-Token: session_xxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Smith"
  }'
```
