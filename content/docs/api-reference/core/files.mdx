# Files

The **File** resource allows you to upload and manage documents, images, and other media associated with your Fleetbase operations.

Files are often attached to other resources. For example, a driver's profile picture, a proof-of-delivery signature image, or a PDF invoice.

## The File Object

```json
{
  "id": "file_xxxxxxxx",
  "name": "signature.png",
  "original_name": "IMG_1234.png",
  "extension": "png",
  "content_type": "image/png",
  "size": 102400,
  "url": "https://s3.amazonaws.com/fleetbase/files/signature.png",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The generated name of the file in storage. |
| `original_name` | string | The original name of the file when it was uploaded. |
| `extension` | string | The file extension (e.g., `png`, `pdf`). |
| `content_type` | string | The MIME type of the file. |
| `size` | integer | The size of the file in bytes. |
| `url` | string | The public URL where the file can be accessed. |

## Upload a File

Uploads a new file to Fleetbase storage.

This endpoint expects a `multipart/form-data` request with the file included in the `file` field.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/files \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -F "file=@/path/to/your/local/file.png"
```

## Retrieve a File

Retrieves the details of an existing File object.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/files/file_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Delete a File

Permanently deletes a File from storage. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/files/file_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Files

Returns a list of your Files. The files are returned sorted by creation date, with the most recently uploaded files appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/files \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
