# Places

A **Place** represents a physical location, such as a customer's home, a warehouse, a store, or a delivery destination.

Places are used extensively across Fleetbase. For example, in [FleetOps](/docs/fleetops), an Order has a pickup Place and a dropoff Place. In [Storefront](/docs/storefront), a Store is associated with a Place.

## The Place Object

```json
{
  "id": "place_xxxxxxxx",
  "name": "Acme Corp Headquarters",
  "street1": "123 Main St",
  "street2": "Suite 400",
  "city": "San Francisco",
  "province": "CA",
  "postal_code": "94105",
  "country": "US",
  "location": {
    "type": "Point",
    "coordinates": [-122.395, 37.793]
  },
  "phone": "+14155551234",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The name of the place (e.g., "Home", "Warehouse A"). |
| `street1` | string | The primary street address. |
| `street2` | string | Secondary address information (e.g., apartment or suite number). |
| `city` | string | The city or locality. |
| `province` | string | The state, province, or region. |
| `postal_code` | string | The ZIP or postal code. |
| `country` | string | The two-letter ISO 3166-1 alpha-2 country code. |
| `location` | object | A GeoJSON Point object representing the exact latitude and longitude. |
| `phone` | string | A phone number associated with the location. |

## Create a Place

Creates a new Place object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/places \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp Headquarters",
    "street1": "123 Main St",
    "city": "San Francisco",
    "province": "CA",
    "postal_code": "94105",
    "country": "US",
    "location": {
      "type": "Point",
      "coordinates": [-122.395, 37.793]
    }
  }'
```

## Retrieve a Place

Retrieves the details of an existing Place.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/places/place_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Place

Updates the specified Place by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/places/place_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+14155559999"
  }'
```

## Delete a Place

Permanently deletes a Place. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/places/place_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Places

Returns a list of your Places. The places are returned sorted by creation date, with the most recently created places appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/places \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
