# Contacts

A **Contact** represents a person or entity that your organization interacts with.

Contacts are often used to represent customers, vendors, or specific individuals at a [Company](/docs/api-reference/core/companies). In [FleetOps](/docs/fleetops), an Order typically has a customer Contact and a facilitator Contact.

## The Contact Object

```json
{
  "id": "contact_xxxxxxxx",
  "name": "Jane Doe",
  "title": "Logistics Manager",
  "email": "jane.doe@example.com",
  "phone": "+14155559876",
  "type": "customer",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The full name of the contact. |
| `title` | string | The job title or role of the contact. |
| `email` | string | The contact's email address. |
| `phone` | string | The contact's phone number. |
| `type` | string | The type of contact (e.g., `customer`, `vendor`, `driver`). |

## Create a Contact

Creates a new Contact object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/contacts \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane.doe@example.com",
    "phone": "+14155559876",
    "type": "customer"
  }'
```

## Retrieve a Contact

Retrieves the details of an existing Contact.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/contacts/contact_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Contact

Updates the specified Contact by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/contacts/contact_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Senior Logistics Manager"
  }'
```

## Delete a Contact

Permanently deletes a Contact. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/contacts/contact_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Contacts

Returns a list of your Contacts. The contacts are returned sorted by creation date, with the most recently created contacts appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/contacts \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
