# Webhooks

The **Webhook** resource allows you to programmatically create and manage webhook endpoints.

While you can manage webhooks through the [Developer Console](/docs/platform/developer-console/webhooks) UI, this API allows you to automate the process (e.g., if you are building an integration that needs to automatically register a webhook when a user connects their account).

## The Webhook Endpoint Object

```json
{
  "id": "webhook_xxxxxxxx",
  "url": "https://your-app.com/webhooks/fleetbase",
  "description": "Production order events",
  "events": [
    "order.created",
    "order.dispatched",
    "order.completed"
  ],
  "status": "active",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `url` | string | The HTTPS URL where Fleetbase will send the webhook payloads. |
| `description` | string | An optional description of what this webhook is used for. |
| `events` | array | A list of event types that this webhook is subscribed to. |
| `status` | string | The status of the webhook (`active` or `inactive`). |

## Create a Webhook Endpoint

Creates a new Webhook Endpoint object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/webhooks \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-app.com/webhooks/fleetbase",
    "description": "Production order events",
    "events": [
      "order.created",
      "order.dispatched",
      "order.completed"
    ]
  }'
```

## Retrieve a Webhook Endpoint

Retrieves the details of an existing Webhook Endpoint.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/webhooks/webhook_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Webhook Endpoint

Updates the specified Webhook Endpoint by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/webhooks/webhook_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "inactive"
  }'
```

## Delete a Webhook Endpoint

Permanently deletes a Webhook Endpoint. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/webhooks/webhook_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Webhook Endpoints

Returns a list of your Webhook Endpoints. The endpoints are returned sorted by creation date, with the most recently created endpoints appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/webhooks \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
