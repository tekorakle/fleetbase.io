# Vehicles

A **Vehicle** represents a physical vehicle in your fleet (e.g., a car, truck, van, or motorcycle).

Vehicles can be assigned to [Drivers](/docs/api-reference/fleetops/drivers) and [Fleets](/docs/api-reference/fleetops/fleets). They can also be connected to telematics devices to receive real-time location and diagnostic data.

## The Vehicle Object

```json
{
  "id": "vehicle_xxxxxxxx",
  "make": "Ford",
  "model": "Transit",
  "year": "2022",
  "plate_number": "ABC-1234",
  "vin": "1FTYR1YV2NKXXXXXX",
  "status": "active",
  "fleet": "fleet_xxxxxxxx",
  "vendor": null,
  "location": {
    "type": "Point",
    "coordinates": [-122.395, 37.793]
  },
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `make` | string | The manufacturer of the vehicle (e.g., "Ford"). |
| `model` | string | The model of the vehicle (e.g., "Transit"). |
| `year` | string | The manufacturing year of the vehicle. |
| `plate_number` | string | The license plate number. |
| `vin` | string | The Vehicle Identification Number. |
| `status` | string | The current status of the vehicle (e.g., `active`, `maintenance`, `inactive`). |
| `fleet` | string | The ID of the [Fleet](/docs/api-reference/fleetops/fleets) the vehicle belongs to. |
| `vendor` | string | The ID of the [Vendor](/docs/api-reference/fleetops/vendors) that owns the vehicle (if outsourced). |
| `location` | object | A GeoJSON Point object representing the vehicle's last known location. |

## Create a Vehicle

Creates a new Vehicle object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/fleet-ops/vehicles \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "make": "Ford",
    "model": "Transit",
    "year": "2022",
    "plate_number": "ABC-1234",
    "status": "active"
  }'
```

## Retrieve a Vehicle

Retrieves the details of an existing Vehicle.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/vehicles/vehicle_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Vehicle

Updates the specified Vehicle by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/fleet-ops/vehicles/vehicle_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "maintenance"
  }'
```

## Delete a Vehicle

Permanently deletes a Vehicle. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/fleet-ops/vehicles/vehicle_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Vehicles

Returns a list of your Vehicles. The vehicles are returned sorted by creation date, with the most recently created vehicles appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/vehicles \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
