# Fleets

A **Fleet** is a logical grouping of [Drivers](/docs/api-reference/fleetops/drivers) and [Vehicles](/docs/api-reference/fleetops/vehicles).

Fleets are useful for organizing your operations by region, vehicle type, or business unit (e.g., "Downtown Delivery Vans", "Long-Haul Trucks").

## The Fleet Object

```json
{
  "id": "fleet_xxxxxxxx",
  "name": "Downtown Delivery Vans",
  "service_area": "area_xxxxxxxx",
  "zone": "zone_xxxxxxxx",
  "task": "delivery",
  "status": "active",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The name of the fleet. |
| `service_area` | string | The ID of the [Service Area](/docs/api-reference/fleetops/service-areas) this fleet operates in. |
| `zone` | string | The ID of the [Zone](/docs/api-reference/fleetops/zones) this fleet operates in. |
| `task` | string | The primary task or purpose of the fleet (e.g., `delivery`, `service`). |
| `status` | string | The current status of the fleet (e.g., `active`, `inactive`). |

## Create a Fleet

Creates a new Fleet object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/fleet-ops/fleets \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Downtown Delivery Vans",
    "task": "delivery",
    "status": "active"
  }'
```

## Retrieve a Fleet

Retrieves the details of an existing Fleet.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/fleets/fleet_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Fleet

Updates the specified Fleet by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/fleet-ops/fleets/fleet_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "inactive"
  }'
```

## Delete a Fleet

Permanently deletes a Fleet. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/fleet-ops/fleets/fleet_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Fleets

Returns a list of your Fleets. The fleets are returned sorted by creation date, with the most recently created fleets appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/fleets \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
