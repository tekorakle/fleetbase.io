# Service Rates

A **Service Rate** defines the pricing structure for a specific type of delivery or service.

Service Rates are used to automatically calculate [Service Quotes](/docs/api-reference/fleetops/service-quotes) and determine the final price of an [Order](/docs/api-reference/fleetops/orders).

## The Service Rate Object

```json
{
  "id": "rate_xxxxxxxx",
  "service_name": "Standard Delivery",
  "service_type": "standard",
  "base_fee": 500,
  "per_meter_flat_rate_fee": 0,
  "per_meter_fee": 0.05,
  "per_minute_flat_rate_fee": 0,
  "per_minute_fee": 0.10,
  "currency": "USD",
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `service_name` | string | The human-readable name of the service rate (e.g., "Standard Delivery"). |
| `service_type` | string | The internal identifier for the service type (e.g., `standard`, `express`). |
| `base_fee` | integer | The fixed base fee applied to every order using this rate (in cents). |
| `per_meter_flat_rate_fee` | integer | A flat fee applied if the distance exceeds a certain threshold (in cents). |
| `per_meter_fee` | number | The fee charged per meter of distance (in cents). |
| `per_minute_flat_rate_fee` | integer | A flat fee applied if the duration exceeds a certain threshold (in cents). |
| `per_minute_fee` | number | The fee charged per minute of duration (in cents). |
| `currency` | string | The currency of the fees (e.g., `USD`). |

## Create a Service Rate

Creates a new Service Rate object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/fleet-ops/service-rates \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "service_name": "Standard Delivery",
    "service_type": "standard",
    "base_fee": 500,
    "per_meter_fee": 0.05,
    "per_minute_fee": 0.10,
    "currency": "USD"
  }'
```

## Retrieve a Service Rate

Retrieves the details of an existing Service Rate.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/service-rates/rate_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Service Rate

Updates the specified Service Rate by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/fleet-ops/service-rates/rate_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "base_fee": 600
  }'
```

## Delete a Service Rate

Permanently deletes a Service Rate. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/fleet-ops/service-rates/rate_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Service Rates

Returns a list of your Service Rates. The rates are returned sorted by creation date, with the most recently created rates appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/service-rates \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
