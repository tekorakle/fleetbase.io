# Service Quotes

A **Service Quote** represents an estimated price for a delivery or service based on the configured [Service Rates](/docs/api-reference/fleetops/service-rates).

You can generate a service quote before creating an order to show the customer the estimated cost.

## The Service Quote Object

```json
{
  "id": "quote_xxxxxxxx",
  "service_rate": "rate_xxxxxxxx",
  "amount": 1500,
  "currency": "USD",
  "distance": 15000,
  "time": 1800,
  "is_estimated": true,
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `service_rate` | string | The ID of the [Service Rate](/docs/api-reference/fleetops/service-rates) used to calculate this quote. |
| `amount` | integer | The total quoted amount (in cents). |
| `currency` | string | The currency of the quote (e.g., `USD`). |
| `distance` | integer | The estimated distance of the route in meters. |
| `time` | integer | The estimated duration of the route in seconds. |
| `is_estimated` | boolean | Whether the quote is an estimate or a fixed price. |

## Generate a Service Quote

Generates a new Service Quote based on the provided parameters.

You must provide either the `pickup` and `dropoff` coordinates, or the IDs of existing [Places](/docs/api-reference/core/places).

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/fleet-ops/service-quotes \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "pickup": "place_xxxxxxxx",
    "dropoff": "place_yyyyyyyy",
    "service_type": "standard"
  }'
```

## Retrieve a Service Quote

Retrieves the details of an existing Service Quote.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/service-quotes/quote_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Service Quotes

Returns a list of your Service Quotes. The quotes are returned sorted by creation date, with the most recently created quotes appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/service-quotes \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
