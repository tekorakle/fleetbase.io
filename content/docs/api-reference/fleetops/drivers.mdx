# Drivers

A **Driver** represents a person who fulfills [Orders](/docs/api-reference/fleetops/orders).

Drivers can be assigned to [Vehicles](/docs/api-reference/fleetops/vehicles) and [Fleets](/docs/api-reference/fleetops/fleets). They typically use the Fleetbase Navigator mobile app to receive dispatch notifications, update order statuses, and transmit their real-time location.

## The Driver Object

```json
{
  "id": "driver_xxxxxxxx",
  "name": "John Smith",
  "email": "john.smith@example.com",
  "phone": "+14155551111",
  "status": "active",
  "vehicle": "vehicle_xxxxxxxx",
  "fleet": "fleet_xxxxxxxx",
  "vendor": null,
  "location": {
    "type": "Point",
    "coordinates": [-122.395, 37.793]
  },
  "created_at": "2023-10-27T10:00:00Z",
  "updated_at": "2023-10-27T10:00:00Z"
}
```

### Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | string | Unique identifier for the object. |
| `name` | string | The full name of the driver. |
| `email` | string | The driver's email address. |
| `phone` | string | The driver's phone number. |
| `status` | string | The current status of the driver (e.g., `active`, `inactive`). |
| `vehicle` | string | The ID of the [Vehicle](/docs/api-reference/fleetops/vehicles) currently assigned to the driver. |
| `fleet` | string | The ID of the [Fleet](/docs/api-reference/fleetops/fleets) the driver belongs to. |
| `vendor` | string | The ID of the [Vendor](/docs/api-reference/fleetops/vendors) the driver works for (if outsourced). |
| `location` | object | A GeoJSON Point object representing the driver's last known location. |

## Create a Driver

Creates a new Driver object.

### Request

```bash
curl -X POST https://api.fleetbase.io/v1/fleet-ops/drivers \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Smith",
    "phone": "+14155551111",
    "status": "active"
  }'
```

## Retrieve a Driver

Retrieves the details of an existing Driver.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/drivers/driver_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## Update a Driver

Updates the specified Driver by setting the values of the parameters passed.

### Request

```bash
curl -X PUT https://api.fleetbase.io/v1/fleet-ops/drivers/driver_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "vehicle": "vehicle_xxxxxxxx"
  }'
```

## Delete a Driver

Permanently deletes a Driver. It cannot be undone.

### Request

```bash
curl -X DELETE https://api.fleetbase.io/v1/fleet-ops/drivers/driver_xxxxxxxx \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```

## List all Drivers

Returns a list of your Drivers. The drivers are returned sorted by creation date, with the most recently created drivers appearing first.

### Request

```bash
curl -X GET https://api.fleetbase.io/v1/fleet-ops/drivers \
  -H "Authorization: Bearer flb_test_xxxxxxxxxxxx"
```
