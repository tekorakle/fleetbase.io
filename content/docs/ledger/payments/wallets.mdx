# Wallets

A **Wallet** is a digital balance account within Ledger. Wallets track money owed to or held for a specific entity — a driver's earnings, a customer's prepaid credit, a vendor's payout balance, or your organization's operating account.

Navigate to **Ledger → Payments → Wallets**.

![Wallet list — owner name, type, currency, balance, status badge](/images/placeholder.png)

## Wallet Types

Wallets are polymorphic — they belong to any Fleetbase entity:

| Type | Owner | Common Use |
| :--- | :--- | :--- |
| **Driver** | Fleet-Ops driver | Track earnings from completed deliveries |
| **Customer** | Storefront customer | Prepaid credit, loyalty balance, refunds |
| **Company** | Your organization | Platform revenue, operating account |
| **User** | Individual Fleetbase user | Personal account |

A single entity can have multiple wallets in different currencies.

## Wallet Status

| Status | Behavior |
| :--- | :--- |
| **active** | Normal — credits and debits allowed |
| **frozen** | Debits blocked; credits still allowed (e.g., during dispute review) |
| **closed** | No operations allowed |

## Creating a Wallet

Wallets are often provisioned automatically — when a new driver is created in Fleet-Ops, or when a customer registers in Storefront. To create one manually:

<Steps>
  <Step>
    Navigate to **Ledger → Payments → Wallets**.
  </Step>
  <Step>
    Click **New Wallet**.
  </Step>
  <Step>
    Select the **Owner** (driver, customer, company, or user).
  </Step>
  <Step>
    Select the **Currency**.
  </Step>
  <Step>
    Enter an optional **Name** and **Description**.
  </Step>
  <Step>
    Click **Save**.
  </Step>
</Steps>

## Wallet Operations

### Deposit (Credit)

Add funds to a wallet without a gateway (e.g., manual credit, earning allocation):

1. Open the wallet
2. Click **Credit**
3. Enter the amount and description
4. Click **Save**

Journal entry created: `DEBIT Cash → CREDIT Wallet Liability`

### Top-Up (Credit via Gateway)

Add funds to a wallet using a payment gateway (e.g., customer tops up with a credit card):

1. Open the wallet
2. Click **Top Up**
3. Select the gateway and enter the amount
4. Click **Charge**

The gateway processes the payment, then credits the wallet.

### Withdrawal (Debit)

Remove funds from a wallet (e.g., driver payout, refund):

1. Open the wallet
2. Click **Withdraw**
3. Enter the amount and description
4. Click **Save**

Journal entry created: `DEBIT Wallet Liability → CREDIT Cash`

<Callout type="warn">
Withdrawals are blocked on frozen wallets. Unfreeze the wallet first if a debit is legitimately needed.
</Callout>

### Transfer

Move funds between two wallets:

1. Open the source wallet
2. Click **Transfer**
3. Select the destination wallet and enter the amount
4. Click **Transfer**

Two transactions are created — a debit from the source and a credit to the destination — with corresponding journal entries.

### Freeze & Unfreeze

To freeze a wallet:
1. Open the wallet
2. Click **Freeze**

To unfreeze: click **Unfreeze**. Only administrators can freeze/unfreeze wallets.

## Viewing Wallet Transactions

Open any wallet and click the **Transactions** tab to see a chronological list of all transactions with amounts, directions, and running balance.

## Balance Reconciliation

If a wallet balance appears incorrect, click **Recalculate Balance** from the wallet detail. This recomputes the balance from scratch based on all completed transactions.
