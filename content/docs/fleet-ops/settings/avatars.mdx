# Avatars

**Avatars** are the profile images and icons used to visually identify drivers, vehicles, contacts, and other resources in the Fleet-Ops console and in the Navigator app. The Avatar Manager provides a centralized library of images you can assign across resources.

Navigate to **Fleet-Ops → Settings → Avatars**.

![Avatar Manager — grid of avatar images with categories for drivers, vehicles, and contacts](/images/placeholder.png)

## Avatar Categories

The avatar library is organized into categories:

| Category | Used for |
| :--- | :--- |
| **Driver** | Driver profile photos and default icons |
| **Vehicle** | Vehicle photos and type icons (car, van, truck, motorcycle) |
| **Contact** | Customer and supplier profile images |
| **Fleet** | Fleet group icons |
| **Place** | Location type icons (warehouse, hub, store) |

## Uploading Avatars

<Steps>
  <Step>
    Navigate to **Fleet-Ops → Settings → Avatars**.
  </Step>
  <Step>
    Select the **category** tab for the resource type you want to add avatars for.
  </Step>
  <Step>
    Click **Upload** and select image files (PNG, JPG, SVG recommended).
  </Step>
  <Step>
    The uploaded images appear in the category gallery and are immediately available for selection on resource records.
  </Step>
</Steps>

## Assigning Avatars to Resources

Avatars are assigned on individual resource records:

- **Driver** — open the driver record → click the avatar area → select from the gallery
- **Vehicle** — open the vehicle record → click the avatar area → select or upload
- **Contact** — open the contact record → upload a profile photo or logo

Selected avatars appear:
- In the drivers and vehicles list view
- On the live map as marker icons (vehicle and driver icons)
- In the Navigator app's driver profile
- In order detail panels next to the assigned driver

## Default Avatars

Fleet-Ops ships with a set of default avatar icons for common vehicle types:
- Sedan / Car
- Van / Minivan
- Truck
- Motorcycle / Scooter
- Bicycle
- Drone

And for drivers:
- Generic person silhouettes (male/female/neutral)
- Numbered icons for quick identification

## Map Marker Icons

Vehicle avatar icons displayed on the live map are the same images from the avatar library, resized for use as map markers. For best results on the map, upload icons with:

- Square dimensions (e.g., 64×64 px)
- Transparent background (PNG or SVG)
- High contrast colors visible on both light and dark map themes

## Deleting Avatars

To remove an avatar from the library, click the image in the gallery and select **Delete**. Deleting an avatar that is currently assigned to a resource will revert that resource to the default icon.
