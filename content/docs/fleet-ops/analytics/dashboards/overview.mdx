# Analytics Dashboards

**Dashboards** provide a live view of your fleet's operational performance through KPI widgets and charts. The Fleet-Ops dashboard shows the metrics most relevant to dispatchers and fleet managers — order volumes, driver status, delivery success rates, and fleet utilization.

Navigate to **Fleet-Ops → Analytics → Dashboards**.

![Fleet-Ops analytics dashboard — KPI cards across the top, charts below showing trends](/images/placeholder.png)

## Default Dashboard Widgets

The default Fleet-Ops analytics dashboard includes:

### Order KPIs
| Widget | What it shows |
| :--- | :--- |
| **Orders Today** | Total orders created today |
| **Orders Completed** | Completed orders for the selected period |
| **Orders Pending** | Orders currently awaiting dispatch |
| **Completion Rate** | Percentage of orders completed vs. total created |
| **Cancellation Rate** | Percentage of orders cancelled |
| **Average Delivery Time** | Mean time from order creation to completion |

### Driver Activity
| Widget | What it shows |
| :--- | :--- |
| **Drivers Online** | Count of drivers currently active in Navigator |
| **Drivers on Active Orders** | Drivers currently executing an order |
| **Top Performing Drivers** | Drivers with the most completions in the period |

### Fleet Status
| Widget | What it shows |
| :--- | :--- |
| **Vehicles Online** | Count of vehicles currently reporting location |
| **Fleet Utilization** | Percentage of vehicles currently assigned to an active order |

### Financial
| Widget | What it shows |
| :--- | :--- |
| **Revenue** | Total value of completed service quotes/transactions |
| **Average Order Value** | Mean revenue per completed order |

## Date Range Filter

Use the date range selector at the top of the dashboard to change the period shown across all widgets:

- Today
- Yesterday
- Last 7 days (default)
- Last 30 days
- Last 90 days
- Custom date range

## Trend Charts

Below the KPI widgets, trend charts show:

- **Orders over time** — daily order volume as a bar chart
- **Completion rate trend** — daily success rate over the period
- **Driver activity** — online hours per driver

Charts are interactive — hover to see daily values, click to drill down.

## Dashboard Customization

Dashboards can be customized to show the widgets most relevant to your team:

1. Click **Customize** in the dashboard toolbar
2. Toggle individual widgets on/off
3. Drag widgets to reorder them
4. Set the default date range for the dashboard

Multiple dashboards can be created for different roles — e.g., a dispatcher dashboard focused on today's orders, and a manager dashboard focused on weekly trends.

## Accessing the Dashboard from Fleet-Ops Home

When you open Fleet-Ops, the main home screen shows a summary dashboard. The full analytics dashboard with all widgets and charts is in **Analytics → Dashboards**.
