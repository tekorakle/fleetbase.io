# Analytics Overview

The **Analytics** section of Fleet-Ops provides visibility into fleet performance through dashboards and reports. Use it to monitor delivery success rates, driver utilization, fleet efficiency, and other operational KPIs.

| Section | Purpose |
| :--- | :--- |
| [**Dashboards**](/docs/fleet-ops/analytics/dashboards/overview) | Pre-built and customizable KPI dashboards |
| [**Reports**](/docs/fleet-ops/analytics/reports/overview) | Custom reports with filters, grouping, and export |

## Key Metrics Available

Fleet-Ops tracks the following metrics across your operation:

### Delivery Performance
- Orders completed per day/week/month
- On-time delivery rate
- Average delivery time (creation to completion)
- First-attempt delivery success rate
- Orders cancelled and reasons

### Driver Performance
- Orders completed per driver
- Average delivery time per driver
- Distance driven per driver
- On-time rate per driver
- Issues reported per driver

### Fleet Utilization
- Active vs. inactive vehicles
- Average vehicle utilization rate
- Distance driven per vehicle
- Fuel consumption per vehicle (from fuel reports)
- Maintenance downtime

### Operational
- Orders by status (pending, dispatched, completed, cancelled)
- Orders by type (order config)
- Revenue (from service quotes/transactions)
- Average order distance

## Accessing Analytics

Navigate to **Fleet-Ops → Analytics** in the left sidebar. The dashboards and reports sub-sections are listed below.

## Data Freshness

Analytics data is derived from the live Fleet-Ops database. Dashboards show current state and recent history. Reports can be generated for any historical date range for which data exists.
