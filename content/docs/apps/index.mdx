# Mobile Apps

Fleetbase provides two fully functional, open-source mobile applications built with React Native (Expo). These apps are designed to work seamlessly with the Fleetbase API and can be customized, white-labeled, and deployed to the App Store and Google Play Store under your own brand.

## The Navigator App

The **Navigator App** is a driver-facing application that integrates with the [FleetOps](/docs/fleetops/overview) extension.

It allows drivers to:
- Receive and accept dispatched orders.
- Navigate to pickup and dropoff locations using turn-by-turn directions.
- Collect proof of delivery (signatures, photos, QR codes).
- Manage their vehicle and shift status.
- Broadcast their real-time location back to the Fleetbase console.

[Learn more about the Navigator App &rarr;](/docs/apps/navigator-app/overview)

## The Storefront App

The **Storefront App** is a customer-facing application that integrates with the [Storefront](/docs/storefront/overview) extension.

It allows customers to:
- Browse products and categories from multiple stores.
- Add items to their cart and proceed to checkout.
- Pay using integrated payment gateways (e.g., Stripe).
- Track their order delivery in real-time on a map.
- Manage their account, addresses, and order history.

[Learn more about the Storefront App &rarr;](/docs/apps/storefront-app/overview)
