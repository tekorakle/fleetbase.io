# Storefront App Overview

The **Storefront App** is an open-source, React Native (Expo) mobile application designed for customers. It integrates directly with the [Storefront](/docs/storefront/overview) extension to provide a complete, white-label e-commerce and on-demand delivery experience.

## Key Features

- **Multi-Store Browsing**: Customers can browse multiple stores, view their operating hours, and see their locations on a map.
- **Product Catalog**: View products, categories, variants, and add-ons.
- **Cart & Checkout**: A seamless shopping cart and checkout flow, including delivery address selection and multiple payment methods (Stripe, Cash on Delivery).
- **Real-Time Order Tracking**: Customers can track their active orders on a live map as the driver approaches, powered by SocketCluster.
- **Order History**: View past orders, reorder favorite items, and leave reviews.
- **Customer Accounts**: Secure authentication, profile management, and saved delivery addresses.
- **Push Notifications**: Receive updates when an order is accepted, dispatched, or completed.

## Supported Platforms

Because the Storefront App is built with React Native and Expo, a single codebase compiles to native applications for both major mobile platforms:

- **iOS**: Requires iOS 13.0 or later.
- **Android**: Requires Android 6.0 (API level 23) or later.

## Integration with Storefront

The Storefront App acts as the customer-facing "client" for the Storefront extension.

When a customer opens the app, it fetches the list of available stores and products via the Storefront API. When they place an order, the app sends a checkout request to the API, which creates an [Order](/docs/api-reference/fleetops/orders) in FleetOps and processes the payment via the [Ledger](/docs/ledger/overview) extension.

As the order progresses through the delivery lifecycle (managed by the dispatcher or the driver using the [Navigator App](/docs/apps/navigator-app/overview)), the Storefront App receives real-time socket events to update the customer's tracking view.
