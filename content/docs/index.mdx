# Introduction

Fleetbase is an open-source, modular logistics operating system built for developers and operators. Whether you are deploying a fleet management platform, building a headless e-commerce marketplace, or developing a custom logistics extension, this documentation covers everything you need.

## Where to Start

- **New to Fleetbase?** Start with [What is Fleetbase?](/docs/getting-started/what-is-fleetbase) and the [Quick Start](/docs/quick-start) guide.
- **Setting up your instance?** See [Deployment Options](/docs/getting-started/deployment-options) and the [Self-Hosted Quickstart](/docs/getting-started/quickstart-self-hosted).
- **Managing a fleet?** Head to the [Fleet-Ops](/docs/fleetops) section.
- **Building a marketplace?** Start with [Storefront](/docs/storefront).
- **Building an extension?** See [Building & Extending Fleetbase](/docs/extending-fleetbase).
- **Integrating via API?** Go to the [API Reference](/docs/api-reference).
