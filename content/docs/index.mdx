# Fleetbase Documentation

Fleetbase is an open-source, modular **Logistics and Supply Chain Operating System** — a production-ready platform that operators and developers can deploy on their own infrastructure, extend with custom logic, and own completely. No vendor lock-in. No per-seat pricing. No black boxes.

## Platform Modules

Fleetbase ships with a set of first-party modules that cover the most common logistics and supply chain use cases. Each module is an independent extension — install what you need, skip what you don't.

| Module | What it does |
| :--- | :--- |
| **[FleetOps](/docs/fleet-ops)** | Fleet management, order dispatch, real-time driver tracking, and route optimisation. The core TMS for delivery and field service operations. |
| **[Storefront](/docs/storefront)** | Headless e-commerce and multi-vendor marketplace engine. Powers consumer-facing ordering apps and hyperlocal delivery platforms. |
| **[Pallet](/docs/pallet)** | Inventory and warehouse management. Track stock across stores, warehouses, and dark stores — integrated with Storefront. |
| **[Ledger](/docs/ledger)** | Accounting and invoicing. Generate invoices from orders and track financials across your operation. |
| **[Developer Console](/docs/platform/developer-console/api-keys)** | API key management, webhook configuration, socket event monitoring, and request log viewing. |

## What the Platform Docs Cover

This **Platform** section covers the core Fleetbase system that underpins every module:

- **Getting Started** — Architecture, deployment options, and orientation
- **Quickstart** — Get running on the cloud or locally in minutes
- **System Setup** — Configure mail, storage, queues, sockets, and branding
- **Identity & Access** — Users, roles, permissions, policies, and organisations
- **Developer Console** — API keys, webhooks, socket events, and request logs
- **Extensions** — Browse and manage extensions from the marketplace
- **Console Features** — Navigation, dashboards, custom fields, internationalisation, and notifications
