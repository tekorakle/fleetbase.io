# Guides & Examples

Welcome to the Fleetbase Guides & Examples section. Here you'll find step-by-step tutorials, real-world use cases, and best practices for building on top of the Fleetbase platform.

Whether you're building a last-mile delivery app, an on-demand marketplace, or integrating with your existing ERP, these guides will help you get up and running quickly.

## Featured Guides

### [Build a Last-Mile Delivery App](/docs/guides/build-a-last-mile-delivery-app)
Learn how to use FleetOps and the Navigator app to build a complete last-mile delivery solution, from order creation to proof of delivery.

### [Build an On-Demand Marketplace](/docs/guides/build-an-on-demand-marketplace)
Discover how to combine Storefront and FleetOps to create a fully functional on-demand marketplace with real-time dispatching.

### [Integrate with your ERP](/docs/guides/integrate-with-your-erp)
A comprehensive guide to syncing orders, inventory, and customers between Fleetbase and your existing ERP system using Webhooks and the Core API.

### [Track a Fleet in Real-Time](/docs/guides/track-a-fleet-in-real-time)
Learn how to use Fleetbase's Socket Events and Telematics integrations to build a real-time fleet tracking dashboard.
