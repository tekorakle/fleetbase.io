# Track a Fleet in Real Time

This guide explains how to use Fleetbase's real-time Socket Events to build a live fleet tracking dashboard or map view.

By subscribing to socket events, you can receive real-time location updates from drivers using the Navigator app or from vehicles equipped with telematics devices.

## Prerequisites

- A Fleetbase account (or a self-hosted instance).
- A **SocketCluster** connection (Fleetbase uses SocketCluster for real-time events).
- A **Public Key** from the [Developer Console](/docs/platform/developer-console/api-keys).

## Step 1: Connect to the Socket Server

First, you need to connect to the Fleetbase SocketCluster server from your frontend application.

You can use the official `socketcluster-client` library for JavaScript:

```javascript
import socketClusterClient from 'socketcluster-client';

const socket = socketClusterClient.create({
  hostname: 'socket.fleetbase.io',
  port: 443,
  secure: true,
  path: '/socketcluster/',
  query: {
    publicKey: 'flb_test_xxxxxxxxxxxx'
  }
});

socket.on('connect', () => {
  console.log('Connected to Fleetbase Socket Server');
});
```

*(If you are self-hosting Fleetbase, replace `socket.fleetbase.io` with your own socket server domain).*

## Step 2: Subscribe to Driver Location Updates

Once connected, you can subscribe to specific channels to receive real-time updates.

To track a specific driver's location, subscribe to the `driver.{driver_id}.location` channel:

```javascript
const driverId = 'driver_xxxxxxxx';
const channel = socket.subscribe(`driver.${driverId}.location`);

channel.watch((data) => {
  console.log(`Driver ${driverId} moved to:`, data.location);
  
  // Update your map marker here
  // updateMarker(driverId, data.location.coordinates);
});
```

The `data` object will contain a GeoJSON Point representing the driver's current location:

```json
{
  "driver_id": "driver_xxxxxxxx",
  "location": {
    "type": "Point",
    "coordinates": [-122.395, 37.793]
  },
  "heading": 90,
  "speed": 45,
  "timestamp": "2023-10-27T14:30:00Z"
}
```

## Step 3: Subscribe to Order Tracking

If you want to track a specific order (e.g., for a customer-facing tracking page), you can subscribe to the `order.{order_id}.tracking` channel.

This channel will emit events whenever the order's status changes or when the assigned driver's location updates.

```javascript
const orderId = 'order_xxxxxxxx';
const channel = socket.subscribe(`order.${orderId}.tracking`);

channel.watch((data) => {
  if (data.type === 'location_update') {
    console.log('Driver location updated:', data.location);
  } else if (data.type === 'status_update') {
    console.log('Order status changed to:', data.status);
  }
});
```

## Step 4: Build the Map View

You can now use a mapping library like Mapbox GL JS, Google Maps, or Leaflet to render the live locations on a map.

Whenever a `location_update` event is received, simply update the coordinates of the corresponding marker on the map.
