# Settings

The **Settings** section of the Storefront Console is where you configure the technical integrations and global preferences that power your e-commerce platform.

## Available Settings

### [API](/docs/storefront/settings/api)
Manage your Storefront API keys. You will need these keys to authenticate requests from your frontend application (e.g., the Storefront App or a custom web app) to the Fleetbase backend.

### [Gateways](/docs/storefront/settings/gateways)
Configure your payment processors. This is where you enter your Stripe credentials to enable secure credit card processing during [Checkout](/docs/storefront/orders/checkout).

### [Locations](/docs/storefront/settings/locations)
Manage global location settings, such as default countries and geocoding preferences.

### [Notifications](/docs/storefront/settings/notifications)
Configure your communication providers. This includes setting up Twilio for SMS OTP authentication and Firebase/APNs for [Push Notifications](/docs/storefront/promotions/push-notifications).

## Accessing Settings

To access these configurations:

1. Log in to the Fleetbase Console.
2. Navigate to the **Storefront** module.
3. Click on **Settings** in the sidebar navigation.
4. Select the specific settings category you wish to configure.

*Note: You must have Administrator privileges within your [Organization](/docs/platform/iam/organizations-and-groups) to view and modify Storefront settings.*
