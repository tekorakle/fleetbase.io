# Promotions & Marketing

The **Promotions** section covers tools to drive customer acquisition, reward repeat buyers, and re-engage your audience.

Navigate to **Storefront → Promotions** to access promotions, push notifications, and review management.

## Discount Codes

Discount codes let customers enter a code at checkout to receive a reduction on their order.

### Creating a discount code

1. Navigate to **Storefront → Promotions**
2. Click **New Promotion**
3. Configure:

| Field | Description |
| :--- | :--- |
| **Name** | Internal reference (e.g., "Summer Sale 2024") |
| **Code** | The code customers enter (e.g., `SUMMER20`). Leave blank for an automated promotion. |
| **Discount Type** | `percentage` (e.g., 20% off) or `fixed` (e.g., $5.00 off) |
| **Discount Value** | The percentage or fixed amount |
| **Start / End Date** | Active window for the promotion |
| **Minimum Spend** | Cart subtotal required to use the code |
| **Usage Limit** | Maximum total uses across all customers |
| **Per-Customer Limit** | How many times one customer can use the code |

4. Click **Save**

### How codes are applied

At checkout, the customer enters the code. Storefront validates:
- Promotion is currently active (within start/end dates)
- Usage limit not reached
- Customer hasn't exceeded per-customer limit
- Cart meets minimum spend requirement

If all checks pass, the discount is applied to the cart subtotal.

## Automated Promotions

Create a promotion **without a code** to make it automatic. Storefront evaluates the rules on every active cart — eligible carts get the discount applied without the customer needing to enter anything.

Use cases:
- Free delivery on orders over $50
- 10% off for all orders during happy hour

## Push Notifications

Send targeted push notifications to customers via APNs (iOS) or FCM (Android). See [Push Notifications](/docs/storefront/promotions/push-notifications) for setup and sending instructions.
