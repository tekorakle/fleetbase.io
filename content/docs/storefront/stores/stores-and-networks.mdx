# Creating Stores & Networks

This guide covers creating and configuring Stores and Networks. For a conceptual overview see [Stores & Networks](/docs/storefront/stores/overview).

## Creating a Store

<Steps>
  <Step>
    Navigate to **Storefront** in the Fleetbase Console sidebar.
  </Step>
  <Step>
    Click **New Store**.
  </Step>
  <Step>
    Enter the **Name** and **Description** for the store.
  </Step>
  <Step>
    Select the **Currency** used for all product pricing in this store.
  </Step>
  <Step>
    Set the **Timezone** — operating hours are evaluated in this timezone.
  </Step>
  <Step>
    Upload a **Logo** (square) and **Backdrop** (wide banner) image for branding.
  </Step>
  <Step>
    Click **Save**. The store is created and assigned a unique API key automatically.
  </Step>
</Steps>

![New store form — name, description, currency, timezone, logo, and backdrop fields](/images/placeholder.png)

After creating the store, add [locations](/docs/storefront/stores/store-locations), set [operating hours](/docs/storefront/stores/operating-hours), and configure [payment gateways](/docs/storefront/settings/gateways).

### Store Contact & Social

In the store's **Details** tab you can also set:
- **Website**, **Email**, **Phone**
- **Facebook**, **Instagram**, **Twitter** handles

These are returned in the public API response so your customer app can display contact and social links.

## Creating a Network

<Steps>
  <Step>
    Navigate to **Storefront** in the console and click **New Network** (or switch to the network view if you manage multiple stores).
  </Step>
  <Step>
    Enter the **Name**, **Description**, **Currency**, and **Timezone**.
  </Step>
  <Step>
    Upload a **Logo** and **Backdrop** image for the network's branding.
  </Step>
  <Step>
    Click **Save**. The network receives its own unique `network_*` API key.
  </Step>
</Steps>

## Adding Stores to a Network

<Steps>
  <Step>
    Open the network and click the **Stores** tab.
  </Step>
  <Step>
    Click **Add Store**.
  </Step>
  <Step>
    Search for and select the stores to add. Optionally assign each store a **Category** within the network (e.g., "Pizza", "Sushi").
  </Step>
  <Step>
    Click **Save**.
  </Step>
</Steps>

Stores can belong to multiple networks. When a customer browses the network, only stores that are currently online and within delivery range appear.

## Managing Store Status

To temporarily disable a store without deleting it:

1. Open the store in the console
2. Toggle **Online** to off

When offline, the store does not appear in network searches and customers cannot place new orders. Existing orders are unaffected.

## Inviting Store Managers

Networks can send invitations to store managers:

1. Open the network → **Invitations** tab
2. Click **Send Invite** and enter the recipient's email or phone
3. The recipient receives a link to set up their store manager account

## Deleting a Store or Network

Open the entity in the console and click **Delete**. Deleting a store removes its catalog — products, categories, and variants. Historical orders are retained for record-keeping. This action cannot be undone.
